corplot(mcmctools)R Documentation

Plot of a correlation matrix

Description

Creates an image plot of a correlation matrix where colors of different shades represent differing levels of correlation. One possible use of this function is to plot the correlation between posterior draws of an MCMC run. Patterns in the plot can aid in constructing a more efficient blocking structure for the MCMC algorithm.

Usage

corplot(mat, col = gray(11:0/11), ...)

Arguments

mat correlation matrix.
col colors to be used in the plot.
... currently not used.

Details

No details.

Value

Creates a plot.
